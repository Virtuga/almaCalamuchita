Módulos

Módulo Tareas
SISTEMA DE GESTIÓN MÓDULO DE USUARIOS Y CLIENTES Gestión CRUD de usuarios internos.
Login seguro con sistema de roles: administrador y vendedor.
Vendedores solo acceden a módulos comerciales (no facturación ni stock completo).
Gestión de clientes registrados (para ventas o consultas desde la web).
SISTEMA DE GESTIÓN MÓDULO DE STOCK Y PRODUCTOS Alta, baja y edición de productos.
Control dinámico de stock con entradas y salidas.
Asociación de productos con proveedores.
Alertas de bajo stock.
SISTEMA DE GESTIÓN MÓDULO DE COMPRAS Registro de compras realizadas (por administrador o vendedor).
Asociación de compras con proveedores.
Remitos de ingreso generados automáticamente.
SISTEMA DE GESTIÓN MÓDULO DE VENTAS Registro de ventas (presenciales o vía web).
Emisión automática de facturas.
Registro de remitos de salida.
Control de métodos de pago utilizados.
SISTEMA DE GESTIÓN MÓDULO DE MARKETING Creación y edición de promociones (por fecha, medio de pago, categoría de productos).
Control de promociones activas.
Conexión con el front público para mostrar descuentos y ofertas destacadas.
SISTEMA DE GESTIÓN MÓDULO DE ENVÍOS Geolocalización de direcciones ingresadas por clientes.
Posibilidad de registrar "domicilios frecuentes".
Gestión de envíos por zonas o repartidores externos.
Módulo Tareas
PÁGINA WEB e-commerce y institucional MÓDULO DE CATÁLOGO DE PRODUCTOS Visualización de productos por categorías.
Filtro por precio, tipo de producto, disponibilidad.
Imágenes de calidad y descripciones breves.
PÁGINA WEB e-commerce y institucional MÓDULO DE COMPRAS ONLINE Carrito de compras.
Login obligatorio mediante cuenta de Google.
Registro automático como cliente tras login.
Integración con pasarela de pagos (MercadoPago o similar).
Dirección de entrega configurable por el cliente.
PÁGINA WEB e-commerce y institucional MÓDULO DE PROMOCIONES Zona destacada para promociones activas.
Promociones destacadas por tiempo limitado.
Compatible con dispositivos móviles.
PÁGINA WEB e-commerce y institucional MÓDULO DE CONTACTO Formulario con campos: nombre, email, mensaje.
Envío automático a casilla de correo del negocio.
Integración con mapa de ubicación de la tienda física.
PÁGINA WEB e-commerce y institucional MÓDULO INSTITUCIONAL Secciones con contenido autoadministrable:
Historia de la tienda.
Misión, visión y valores.
Fotografías antiguas o del proceso artesanal.
Identidad visual coherente con la estética regional.
